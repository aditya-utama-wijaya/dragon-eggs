   - Large Chinese character displayed
   - Pinyin pronunciation shown

   - Stars earned displayed (3 stars)

6. **Lesson Content**
   - Mix of Chinese characters
   - English translations
   - Colorful, kid-friendly design

### Screenshot Tips
- Use bright, high-contrast colors
- Include UI elements (buttons, progress bars)
- Show actual app functionality
- No added text overlays needed (app is self-explanatory)
- Consistent styling across all screenshots
- Show app in "completed" state (looks polished)

1. **Use all available characters** in descriptions

4. **Update regularly** to show app is maintained
